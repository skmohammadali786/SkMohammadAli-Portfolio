## Project Summary
A premium 3D portfolio website for SK Mohammad Ali, an Engineering Student and Full-Stack Developer. The site features a minimal futuristic, cinematic aesthetic with 3D interactive elements, glassmorphism UI, and smooth animations.

## Tech Stack
- **Frontend**: Next.js 15 (App Router), TypeScript, Tailwind CSS
- **3D Graphics**: Three.js, @react-three/fiber, @react-three/drei
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Styling**: Tailwind CSS with custom charcoal/neon blue theme

## Architecture
- `src/components/3d`: Interactive 3D scenes and objects
- `src/components/ui`: Reusable glassmorphism and UI components
- `src/lib/data`: Centralized portfolio content
- `src/lib/utils`: Styling and helper utilities

## User Preferences
- **Theme**: Minimal futuristic, charcoal black (#121212) base
- **Accent**: Soft neon blue / violet (#00f2ff)
- **Visuals**: Matte surfaces, studio lighting, glassmorphism textures
- **Perspective**: Centered, slightly angled for depth

## Project Guidelines
- No harsh contrast in lighting
- No noisy detail levels
- No logos unless abstract or user-provided
- Clean, unreadable abstract text for thumbnails
- 1024x1024 assets for icons, 2048x2048 for backgrounds

## Common Patterns
- Glassmorphism panels for section backgrounds
- 3D Badges for skills
- Floating glass screens for project thumbnails
- Circular glowing progress ring for loading
